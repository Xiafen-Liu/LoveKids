Project introduction

This project is for the development of an online shopping website called LoveKids 
that sells clothes for children aged 0-12. It allows customers to browse products by 
categories, add products to cart, view, delete, empty and check out cart. 
It also allows user to sign in or sign up as member.

Why the project is useful
It aims at making navigation between and interaction with webpages intuitive 
and easy for customers for a positive shopping experience.

How users can get started with the project
Users can start by running the project in visual studio code.
